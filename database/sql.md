### 创建数据库

```sql
CREATE DATABASE transinfo;
```

### 创建数据表

##### 传感器表

```sql
CREATE TABLE station
(
    SensorID  bigint,
    Latitude  double,
    Longitude double
);
```

#### 传感器位置表

```sql
CREATE TABLE sensorStreet
(
    sensorID varchar(10),
    street   varchar(64)
)
```
